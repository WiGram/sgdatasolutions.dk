Cloud Data Platform for Property Portfolio Management

Designing and implementing a scalable Azure-based data platform for public sector property and facility management

Project Start

Data model design and initial architecture

October 2023

Data Integration

Property data integration pipeline

Ongoing

Overview

Leading the development of a comprehensive data platform for public sector property portfolio management. The solution integrates multiple data sources into a unified data model based on IFC standards, enabling efficient property and facility management across the organization.

Key Responsibilities

  • Designed flexible and scalable data model aligned with IFC standards
  • Developed end-to-end integration pipeline for public property data
  • Implemented medallion architecture for data processing
  • Contributed to Autodesk Revit integration development
  • Lead role in FastAPI data exchange application development

Technical Implementation

  • Built Azure Data Factory pipelines supporting full, incremental, and delta loads
  • Developed SQL Server stored procedures for data transformation
  • Implemented field-to-record mapping system for flexible data ingestion
  • Created data processing layers following medallion architecture principles
  • Designed REST API endpoints for data exchange using FastAPI

Architecture Highlights

  • Azure Cloud-based solution with scalable infrastructure
  • Multi-layer data processing following medallion architecture
  • Flexible data model supporting multiple domains
  • Automated orchestration through Azure Data Factory
  • Modern API layer for data exchange
