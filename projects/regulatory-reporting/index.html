Overview

Had a integral role in leading a successful migration of critical regulatory reporting from legacy mainframe systems to a modern Enterprise Data Warehouse solution. The project served multiple Nordic financial institutions, handling sensitive financial data affecting millions of customers. As the technical lead, I managed the entire data transformation lifecycle while ensuring continuous compliance with multiple regulatory frameworks.

Project Duration: 19 months with two successful production releases

Key Responsibilities

  • Architected and implemented a comprehensive data mart solution on the Enterprise Data Warehouse platform
  • Analyzed and interpreted regulatory reporting requirements from various financial authorities
  • Created detailed object documentation and technical specifications for distinct regulatory reports related to personal and corporate business reporting
  • Designed and implemented complex table schemas in Teradata SQL
  • Mapped and transformed source data from hundreds of tables into a standardized reporting format
  • Developed automated data processing workflows for various reporting frequencies (monthly, quarterly, bi-annual, and annual)
  • Implemented period-specific data controls to ensure accurate reporting timeframes

Technical Implementation

  • Designed dimensional data models and fact tables for centralized reporting data mart
  • Created report-specific denormalized tables optimized for regulatory submissions
  • Implemented robust data transformation pipelines using Teradata SQL
  • Established comprehensive data reconciliation processes
  • Built flexible mapping interfaces for reporting specialists
  • Enhanced mappings to feed into the data mart, enabling non-technical staff to manage reporting mappings

Results & Impact

  • Successfully migrated from mainframe to modern EDW solution
  • Enabled non-technical staff to manage reporting mappings
  • Achieved compliance with multiple regulatory frameworks
  • Established reconciliation with existing financial systems

  • Compliance validation
  • Technical specification reviews
